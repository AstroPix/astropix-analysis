# Deprecated astropix-analysis scripts
Scripts in this folder have been depricated due to compatability with software/firmware versions that are no longer supported, namely astropix-python. 

If you are still taking AstroPix V3 or V4 data with astropix-python, it can be decoded with New_Source_Decoder.py. Other files that New_Source_Decoder.py is dependant on have been depricated. Scripts that are dependent on New_Source_Decoder.py have been depricated.
